# Flutter OS Portfolio

A futuristic portfolio website that simulates an Operating System in the browser.

## Features
- **Virtual Desktop Environment**: Draggable windows, taskbar, and dynamic wallpapers.
- **Terminal**: Interactive command-line for "About Me".
- **App Store**: Showcase of projects with "install" mechanics.
- **System Monitor**: Visualizing skills as system processes.
- **Cyber-Noir Aesthetic**: Deep dark themes with neon pink accents.

## Getting Started

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run on Chrome:
   ```bash
   flutter run -d chrome
   ```

## Architecture
- **State Management**: Riverpod
- **Backend**: Firebase (Configured for future integration)
- **UI**: Custom Glassmorphism widgets
